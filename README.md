# BJUT_notification_distribute
BJUT notification (my.bjut.edu.cn) distributer
Check and push notification to WeChat  
Require:  
- Python 3.x  
- requests  
- lxml  
- re  
- newspaper3k  
  
Useage:  
- Rename `userinfo.sample.py` to `userinfo.py`  
- Write config into `userinfo.py`  
- Run `noti.py`
